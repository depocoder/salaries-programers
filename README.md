# Check_your_payday
 
## Описание проекта.   
Этот проект позволяет узнать среднюю зарплату программистов в Москве через API: HeadHunter и SuperJob
   
## Подготовка к запуску.  
Установить библиотеки командой.  
```
pip install -r requirements.txt
```
        
## Запуск кода.  
```
python3 main.py
```
